The [CLPC](https://drive.google.com/drive/folders/1nylXRKlpB9nIfnUL9CXDxr3xWRGGUw2F?usp=drive_link) dataset is used to evaluate the generalization effect of different coloring models.  
We use the [Qwen3-vl-plus](https://bailian.console.aliyun.com/) model to generate descriptions for the CLPC dataset and manually proofread the generated textual descriptions.
